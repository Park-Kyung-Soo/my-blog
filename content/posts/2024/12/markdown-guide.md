# 마크다운 작성 가이드

이 글에서는 블로그에서 마크다운을 사용하는 방법을 설명합니다.

## 제목 (Headings)

```markdown
# H1 제목
## H2 제목
### H3 제목
#### H4 제목
```

## 텍스트 스타일

- **굵은 글씨**: `**텍스트**`
- *기울임*: `*텍스트*`
- ~~취소선~~: `~~텍스트~~`
- `인라인 코드`: \`코드\`

## 목록

### 순서 없는 목록

- 항목 1
- 항목 2
  - 중첩 항목 2-1
  - 중첩 항목 2-2
- 항목 3

### 순서 있는 목록

1. 첫 번째
2. 두 번째
3. 세 번째

## 링크와 이미지

```markdown
[링크 텍스트](https://example.com)
![이미지 설명](이미지경로.jpg)
```

## 표 (Table)

| 이름 | 나이 | 직업 |
|------|------|------|
| 홍길동 | 25 | 개발자 |
| 김철수 | 30 | 디자이너 |
| 이영희 | 28 | 기획자 |

## 코드 블록

여러 언어를 지원합니다:

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>예시</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
```

```css
.container {
  max-width: 720px;
  margin: 0 auto;
  padding: 2rem;
}
```
