# 안녕하세요! 👋

새로운 블로그를 시작하게 되어 기쁩니다. 이 블로그는 **정적 사이트 생성기**로 만들어졌으며, GitHub Pages에서 호스팅됩니다.

## 이 블로그의 특징

- 📝 **마크다운**으로 글을 작성합니다
- 🏷️ **태그**와 **카테고리**로 글을 분류할 수 있습니다
- 📅 월별로 글을 관리합니다
- 🚀 빠르고 가벼운 정적 페이지입니다

## 코드 하이라이팅 예시

JavaScript 코드 예시입니다:

```javascript
function greeting(name) {
  console.log(`안녕하세요, ${name}님!`);
  return {
    message: '환영합니다',
    timestamp: new Date()
  };
}

greeting('방문자');
```

Python 코드도 지원합니다:

```python
def fibonacci(n):
    """피보나치 수열을 생성합니다."""
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# 처음 10개의 피보나치 수
print(list(fibonacci(10)))
```
