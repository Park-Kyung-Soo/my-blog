# 크리스마스 이브에 하는 회고 🎄

오늘은 2025년 12월 24일, 크리스마스 이브입니다.

창 밖으로 내리는 눈을 보며 (내리고 있다면요 😄), 올 한 해를 돌아보는 시간을 가져봅니다.

## 올해 이룬 것들

### 1. 블로그 시작 🚀

드디어 나만의 블로그를 시작했습니다! 직접 정적 사이트 생성기를 만들어서 운영하고 있어요. 남들이 만들어 놓은 것을 쓰는 것도 좋지만, 직접 만든 도구로 글을 쓰는 기분은 정말 특별합니다.

```javascript
// 나만의 블로그 빌더
const myBlog = {
  created: '2025',
  powered: 'Custom Static Site Generator',
  feeling: '뿌듯함 100%'
};
```

### 2. 새로운 것들을 배웠습니다 📚

- Node.js로 빌드 시스템 만들기
- 마크다운 파싱과 렌더링
- CSS 변수를 활용한 다크모드 구현
- GitHub Actions를 이용한 자동 배포

### 3. 작은 습관들의 변화 🌱

> "작은 것들이 모여 큰 변화를 만든다"

매일 조금씩, 꾸준히 무언가를 해나가는 것의 힘을 느꼈습니다.
